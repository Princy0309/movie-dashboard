# Movie Dashboard

A React application that lets users search for movies, see details, and manage a personal watchlist.

## Features
* Search for any movie using the TMDB API.
* Add or remove movies from a personal Watchlist.
* Statistics tracker to see your average movie rating and favorite genres.
* Light/Dark theme toggle.
